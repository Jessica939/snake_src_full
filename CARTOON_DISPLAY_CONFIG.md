# 🎨 漫画显示配置说明

## 📋 当前漫画配置

### 🎯 关卡胜利后漫画展示

| 关卡 | 触发时机 | 显示漫画 | 说明 |
|------|---------|----------|------|
| **第0关** | 序章 (prologue) | `0_0.png` → `0_1.png` | 序章漫画，两张连续 |
| **第1关** | 胜利后 (victory) | `1_0.png` | 第一关胜利漫画 |
| **第2关** | 胜利后 (victory) | `2_0.png` | 第二关胜利漫画 |
| **第3关** | 胜利后 (victory) | `3_0.png` | 第三关胜利漫画 |
| **第4关** | 胜利后 (victory) | `4_0.png` → `4_1.png` | 第四关胜利漫画，两张连续 |
| **第5关** | 胜利后 (victory) | `5_0.png` → `5_1.png` → `5_1'.png` → `5_2.png` | 结局漫画，四张连续 |

## 🔧 技术实现

### 🎭 显示框分离设计
- **文字框**：`800×430` 位置 `(370, 280)` - 专门显示剧情文字
- **漫画框**：`600×1000` 位置 `(468, 30)` - 专门显示漫画图片
- **智能切换**：自动在文字和漫画之间切换显示

## ⚙️ 配置修改方法

### 添加新漫画
在 `showCartoonForLevel()` 函数中添加新配置：
```cpp
else if (level == X && trigger == "victory") {
    // 第X关胜利后
    m_cartoonPaths << getCartoonPath("X_0.png");
    m_cartoonPaths << getCartoonPath("X_1.png");  // 可选：多张
}
```

### 调整显示框尺寸和位置
```cpp
// 文字框
m_scrollArea->setFixedSize(宽, 高);
m_scrollArea->move(X, Y);

// 漫画框  
m_cartoonLabel->setFixedSize(宽, 高);
m_cartoonLabel->move(X, Y);
```

### 调整漫画大小
```cpp
// 在 showCartoonImage() 函数中
int maxWidth = qMax(600, availableSize.width() - 120);
int maxHeight = qMax(1400, availableSize.height() - 120);
```

## 🎮 游戏体验

1. **序章**：游戏开始时显示 0_0.png → 0_1.png
2. **关卡胜利**：每关完成后自动显示对应编号的漫画
3. **多张连续**：第4关和第5关有多张漫画连续播放
4. **无缝切换**：漫画和文字之间自动切换，不显示额外提示

## 📝 注意事项

- 漫画文件为1024×1536的竖版格式
- 支持PNG格式
- 多张漫画按顺序连续播放
- 点击可切换到下一张漫画
- 按ESC可跳过当前漫画序列
